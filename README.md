# greating
- 자신의 취향대로 DIY(Do It Yourself) 도시락을 구성하고, 실제 판매 중인 식단이 되기까지의 일련의 과정들을 자동화하는 맞춤 도시락 구성 서비스

### 주요 기능
- 이용자는 카테고리별 다른 회원들의 DIY 도시락을 보고, 선호하는 도시락에 투표할 수 있다.
- 상위 투표율 DIY 도시락은 관리자를 통해 실제 판매하는 식단이 될 수 있다.
